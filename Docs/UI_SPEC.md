# UI Specification (UI_SPEC)

本ドキュメントは、X-Display (iPad Client) および AIP Viewer におけるUI設計、画面遷移、インタラクション仕様を定義します。

---

## 1. 画面構成と仕様

### 1.1 Device Browser (ホスト探索・接続画面)
- **役割**: ネットワーク上の送信側ホストデバイス（Mac等）を自動検出し、一覧表示する。
- **UIコンポーネント**:
  - デバイスリスト: 検出されたホスト名、IPアドレス、接続状況（未接続/接続中/切断）を表示。
  - ゼロ構成接続ステータス: 自動検出・自動接続の進行状況をインジケータで表示。
  - 設定ボタン: Settings画面への遷移。
- **インタラクション**:
  - ホストをタップすると接続シーケンスを開始し、必要に応じてPIN入力ポップアップを起動。

### 1.2 PIN Authentication (認証ポップアップ/ビュー)
- **役割**: ホスト接続時のセキュリティPIN認証を行う。
- **UIコンポーネント**:
  - 4〜6桁のPIN入力フィールド（セキュア入力、自動フォーカス）。
  - キーボードと連動したスムーズなレイアウト。
- **インタラクション**:
  - 遷移ヒッチ（もたつき）を徹底的に排除した高速アニメーション。
  - 入力完了時に自動送信および認証処理の実行。

### 1.3 Stream Viewer (ストリーミング表示画面)
- **役割**: `MetalRendererView` を介して、ホストからの画面を低遅延で描画する。
- **UIコンポーネント**:
  - メインレンダリング領域: アスペクト比を維持したまま全画面表示。
  - `TouchOverlayView`: タッチ、ドラッグ、ピンチなどのジェスチャをキャプチャし、ホストへ送信。
- **インタラクション**:
  - 低遅延描画を最優先。UI要素のオーバーレイは最小限とし、ジェスチャのレスポンスを最大化。

### 1.4 Settings View (設定画面)
- **役割**: 画質、ネットワーク、NAS等の各種設定を管理する。
- **UIコンポーネント**:
  - ストリーム設定: ビットレート、FPS上限、デコードモード（H.264/HEVC）。
  - NAS接続管理:
    - WebDAV, SMB (v1/v2/v3), DLNA の接続先設定・追加。
  - 接続モード切替: USB優先、Wi-Fi優先のトグル。

---

## 2. デザインシステム & ガイドライン

### 2.1 ビジュアルデザイン
- **テーマ**: Sleek Dark Mode (ダークカラーベース)
- **カラーパレット**:
  - Background: 深みのあるダークグレー / ブラックベース (`#121212`)
  - Accent: 視認性の高いバイブラントカラー (例: ネオンブルー、ソフトパープル)
- **タイポグラフィ**: システム標準のフォント（San Francisco）を使用し、動的なサイズ変更に対応。

### 2.2 インタラクション & アニメーション
- **トランジション**: 画面遷移やポップアップ表示は、iOS標準の心地よいイージング (Ease-In-Out) を適用。
- **フィードバック**: ボタンタップ時のハプティクスフィードバック（iPad/iPhone対応）による操作感の向上。
- **マイクロアニメーション**: ローディングインジケータや接続中のパルスアニメーションなど、システムの「生存状態」を直感的に伝える。
